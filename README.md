## General
Esta notebook es un intento inicial de scrapear el Boletín Oficial de la República Argentina para analizarlo usando Bag od Words

## Propósitos
Es un repo exploratorio para evaluar la aplicación de BOW en documentos oficiales.

## Cómo usarlo
El script fue generado para correr en el contexto de Google Colab. 
Copiando la notebook a cualquier google drive deberia funcionar.
Funciona localmente también, pero quizás requiera algún minimo ajuste de sintaxis o uso de otras librerías
